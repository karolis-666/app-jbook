# App-Jbook

Online code editor

## Installation

App-Jbook requires [Lerna](https://lerna.js.org/) to be installed globally of exact **@3.22.1** version:
`npm install -g --save-exact@3.22.1`

Install **node_modules** for each package:
`lerna bootstrap`

## Running

To run application run:
`npm run start`
